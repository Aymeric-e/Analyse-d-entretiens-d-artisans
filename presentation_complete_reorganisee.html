Objectifs & Approche

4 Phases principales

  1. Phase 0 : Nettoyage - Prétraitement & segmentation des entretiens
  2. Phase 1 : Détection d'outils - Identifier présence/absence outils par phrase
  3. Phase 2 : Verbalisation - Prédiction difficulté verbalisation (score 0-10) avec BERT vs Ridge
  4. Phase 3 : 6 Facteurs - Modèles supplémentaires + analyse SHAP & corrélations

Phase 0 : Nettoyage & Prétraitement

Pipeline Nettoyage

1️⃣ Extraction Artisans

Supprimer interventions interviewer, garder parole artisans uniquement

2️⃣ Segmentation

Diviser en paragraphes (existants) → puis en phrases (regex.split)

3️⃣ Nettoyage Texte

Supprimer accents, ponctuation extraneous, normaliser espaces

4️⃣ CSV Structurés

Générer 3 niveaux : fiche, paragraphe, phrase

Résultats & Défis

✓ Obtenu

50+ entretiens nettoyés = ~3500 phrases

Corpus homogène & versioning GitHub

⚠️ Défis

• Qualité transcription inégale (accents régionaux)

• Phrases chevauchantes (oral ≠ écrit)

• Ambiguïtés segmentation (où couper?)

Phase 1 : Détection d'Outils

Démarche

❌ NER Pré-entraîné

CamemBERT, FlauBERT → Inefficace : types limités (personne, org, lieu)

✓ Solution : Dictionary Matching

1. Extraction liste d'outils depuis Wikipedia

2. Comparaison stricte texte/dictionnaire

3. Score binaire par phrase

Exemple

"J'utilise le marteau pour taper sur le métal"

✓ Outil trouvé

marteau ∈ dictionnaire

Résultat: Phrase = [1, "marteau"]

Phase 2a : Prédiction Verbalisation

Pipeline & Méthodologie

1. Annotation Manuelle

~500 phrases annotées (score difficulté verbalisation 0-10)

2. Data Augmentation

Back-translation, contextual embedding (nlpaug)

3. Fine-tuning BERT

BERT multilingue → Tête régression [0, 10]

4. Baseline (Régression Ridge)

TF-IDF → Ridge → Comparaison BERT vs classique

Résultats & Performance

Métrique BERT Ridge
R² Score 0.818 0.635
MAE 0.176 0.321
RMSE 0.321 0.454

💡 Insight

BERT +28% meilleur R² que Ridge → Capture contexte sémantique

Phase 2b : Fine-tuning & Défis

Hyperparamètres Optimisés

Paramètre Valeur
Learning Rate 2e-5
Batch Size 16
Epochs 10
Max Length 128
Dropout 0.3
Weight Decay 0.01

Optimizer: AdamW + linear warmup/decay

Défis & Solutions

⚠️ Overfitting Dataset

Petit corpus (~500 phrases) → Dropout 0.3 + Early stopping validation loss

⚠️ Subjectivité Annotation

Accord inter-annotateurs κ = 0.65 → Data augmentation compense

⚠️ Déséquilibre Distribution

Scores concentrés 5-7/10 → Weighted loss par classe
